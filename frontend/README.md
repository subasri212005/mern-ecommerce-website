# 🛒 MERN E-COMMERCE PROJECT

A full-stack MERN Stack E-Commerce web application built for learning and practice.  
It provides product listing, product details, and order management using MongoDB, Express, React, and Node.js.

---

## 🚀 Features

- View all products
- View single product details
- Create orders
- REST API backend
- React frontend integration
- MongoDB database connection
- Production build support

---

## 🧱 Tech Stack

**Frontend:**
- React.js
- Axios
- React Router DOM
- CSS

**Backend:**
- Node.js
- Express.js
- MongoDB + Mongoose
- CORS
- dotenv
